## <a name="whats-translated"></a>翻译的内容

应用使用您的语言首选项以您的首选语言显示元素，例如菜单、标签文本和系统消息。

由于大多数功能基于您的数据，所以某些项仍然是其原始语言。 例如：

- 您的数据，包括实体和细分
- 您的环境名称

## <a name="choose-your-language-settings"></a>选择您的语言设置  

[在 **系统** 页面上](system.md)更改您的首选语言。
